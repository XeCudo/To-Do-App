# To-Do-App
This is simple To Do app with html &amp; css &amp; javaScript | I hope it is useful for you


## Features

- Have simple & user friendly ui
- Responsive design
- Tasks save on local storage and are not deleted by exiting the browser or refreshing
- Have dark mode(Soon)
